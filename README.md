# Age Of King

2D Turn Base, Strategy Game

1.  OOP
2.  Polymorphism
3.  Inheritance
4.  S-O-L-I-D
5.  Design Patterns
     -Factory
     -Singleton
6.  UI and Logic separated from each other using techniques like MVC/MVP.
7.  Draw Call (max 20)
8.  Object Pooling
9.  Coroutine
10. Events
11. Infinite Scrollview with Object Pooling
